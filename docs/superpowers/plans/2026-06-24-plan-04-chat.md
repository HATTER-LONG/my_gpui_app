# Plan 4: 对话面板

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现右侧对话面板——320px 宽滑入/滑出、消息气泡（用户/AI 角色标识）、多行输入框、对话命名、举报剧透按钮。

**Architecture:** `ChatPanel` 持有 `Vec<Message>` 和输入状态。面板宽度通过 `is_visible` bool 控制。消息气泡左右对齐（用户右/ AI 左），每条消息携带角色和人格模式标签。输入框底部固定。

**Tech Stack:** Rust + GPUI

**Dependencies:** Plan 1 (AppState 的 chat_panel_visible 和聊天面板占位), Plan 3 (阅读视图文字选中能力，用于"发起讨论")

**Global Constraints:** 面板宽 320px, 消息圆角 12px, AI 消息右下角 `⚑ 举报剧透` 链接

- [ ] **Step 1: 创建 `src/chat.rs`**

```rust
use gpui::{Context, Window, div, px, prelude::*, IntoElement, ParentElement, Render, Styled};

#[derive(Clone)]
pub enum ChatRole {
    User,
    AI,
}

#[derive(Clone)]
pub enum PersonaMode {
    Professor,
    BookFriend,
}

impl PersonaMode {
    pub fn label(&self) -> &str {
        match self {
            PersonaMode::Professor => "教授模式",
            PersonaMode::BookFriend => "书友模式",
        }
    }
}

pub struct Message {
    pub role: ChatRole,
    pub content: String,
    pub persona: Option<PersonaMode>,
    pub spoiler_reported: bool,
}

pub struct ChatPanel {
    pub messages: Vec<Message>,
    pub input_text: String,
    pub title: String,
    pub current_persona: PersonaMode,
}

impl ChatPanel {
    pub fn new(title: String) -> Self {
        Self {
            messages: Vec::new(),
            input_text: String::new(),
            title,
            current_persona: PersonaMode::Professor,
        }
    }

    pub fn add_message(&mut self, role: ChatRole, content: String) {
        self.messages.push(Message {
            role,
            content,
            persona: Some(self.current_persona.clone()),
            spoiler_reported: false,
        });
    }
}

impl Render for ChatPanel {
    fn render(&mut self, _window: &mut Window, cx: &mut Context<Self>) -> impl IntoElement {
        div()
            .flex()
            .flex_col()
            .size_full()
            .bg(gpui::rgba(0x2d2d2dff))
            .child(self.render_header(cx))
            .child(self.render_message_list(cx))
            .child(self.render_input_area(cx))
    }
}
```

### Task 3: 实现消息气泡渲染

**Files:**
- Modify: `src/chat.rs`

- [ ] **Step 1: 添加消息列表和气泡渲染方法**

```rust
impl ChatPanel {
    fn render_message_list(&self, cx: &mut Context<Self>) -> impl IntoElement {
        div()
            .flex_grow()
            .overflow_y_scroll()
            .px_3()
            .py_2()
            .flex()
            .flex_col()
            .gap_3()
            .children(self.messages.iter().map(|msg| self.render_bubble(msg)))
    }

    fn render_bubble(&self, msg: &Message) -> impl IntoElement {
        let is_user = matches!(msg.role, ChatRole::User);
        let bubble_bg = if is_user {
            gpui::rgba(0x3a3a3aff)
        } else {
            gpui::rgba(0x383838ff)
        };

        div()
            .flex()
            .flex_col()
            .when(is_user, |d| d.items_end())
            .child(
                // 角色标签
                div()
                    .text_xs()
                    .text_color(gpui::rgba(0x888888ff))
                    .mb_1()
                    .child(match msg.role {
                        ChatRole::User => "👤 你".to_string(),
                        ChatRole::AI => {
                            let persona = msg.persona.as_ref()
                                .map(|p| p.label())
                                .unwrap_or("");
                            format!("📖 AI · {}", persona)
                        }
                    }),
            )
            .child(
                // 消息气泡
                div()
                    .max_w(px(272.0)) // 320 * 0.85
                    .px_3()
                    .py_2()
                    .bg(bubble_bg)
                    .rounded(px(12.0))
                    .child(msg.content.clone()),
            )
            .child(
                // AI 消息的举报按钮
                if matches!(msg.role, ChatRole::AI) {
                    div()
                        .text_xs()
                        .text_color(gpui::rgba(0x888888ff))
                        .cursor_pointer()
                        .child(if msg.spoiler_reported {
                            "⚑ 已举报"
                        } else {
                            "⚑ 举报剧透"
                        })
                } else {
                    div()
                },
            )
    }
}
```

```rust
    fn render_input_area(&self, cx: &mut Context<Self>) -> impl IntoElement {
        div()

            .flex_col()
            .px_3()
            .py_2()
            .border_t_1()

                    .max_h(px(100.0))
                    .px_3()
                    .py_2()
                    .bg(gpui::rgba(0x1e1e1eff))
                    .rounded_md()
                    .text_sm()
                    .text_color(if self.input_text.is_empty() {
                        gpui::rgba(0x666666ff)

                        gpui::rgba(0xe0e0e0ff)
                    })

                        self.input_text.clone()
```

```rust
fn render_chat_panel(&self, cx: &mut Context<Self>) -> impl IntoElement {
    if !self.chat_panel_visible {
        return div();
    }
    if let Some(ref panel) = self.chat_panel {
        div()
            .w(px(320.0))
            .h_full()
            .child(panel.render().into_any_element())
    } else {
        div()
            .w(px(320.0))
            .h_full()
            .bg(gpui::rgba(0x2d2d2dff))
            .flex()
            .items_center()
            .justify_center()
            .child("尚未开始对话")
    }
}
```
