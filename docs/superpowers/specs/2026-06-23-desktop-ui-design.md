# AI 阅读陪伴助手 — 桌面端 UI 设计文档

> 版本: 1.0 | 日期: 2026-06-23 | 状态: 已确认
>
> 基于产品需求文档 (spec.md) 的 UI 层面设计细化。

---

## 1. 总体布局方案

采用**经典三栏式**结构，左侧固定导航栏 + 中间主内容区 + 右侧对话面板（按需出现）。

```
┌──────────────────────────────────────────────────────────┐
│  顶部标题栏（系统原生 / 自定义）                 — □ ✕    │
├──────┬─────────────────────────────────────┬─────────────┤
│      │                                     │  对话面板    │
│ 左侧  │           主内容区                  │  (0/320px)  │
│ 导航  │     (书库/阅读/词汇本/设置)          │  按需滑出   │
│ 图标  │                                     │             │
│ 栏   │                                     │             │
│      │                                     │             │
├──────┴─────────────────────────────────────┴─────────────┤
│  底部状态栏（可选，h=24px）                                │
└──────────────────────────────────────────────────────────┘
```

**尺寸规范**：
- 窗口最小宽度：900px
- 左侧导航栏：固定 56px
- 右侧对话面板：0 或 320px（动画过渡 200ms）
- 底部状态栏：24px（MVP 可简化）

**动画**：对话面板展开/收起使用宽度过渡动画 200ms，主内容区同步压缩/扩展。

## 2. 左侧导航栏

固定 56px 宽，`bg(secondary)`，包含 4 个图标项（第 2 项动态出现）。

| 序号 | 图标 | 名称 | 行为 |
|------|------|------|------|
| 1 | 📚 | 书库 | 默认激活项，显示封面网格 |
| 2 | 📖 | 当前阅读 | **动态出现**——仅当有书被打开且未读完时显示；高亮表示"正在读"；点击回到上次阅读位置 |
| 3 | 📝 | 词汇本 | 展示所有书的词汇集合 |
| 4 | ⚙ | 设置 | API Key 配置、对话语言偏好、数据导出/导入 |

**交互**：
- 选中态：左侧 3px 彩色竖条 + 图标高亮
- "当前阅读"项：若用户关闭/读完所有书，该项消失，导航退回到仅 3 项
- 点击导航图标 → 主内容区切换到对应视图；若右侧对话面板处于展开状态，自动收起

## 3. 书库视图

### 3.1 空状态（首次使用）

```
┌─────────────────────────────────────────┐
│                                         │
│           📖  (大图标，160px)            │
│                                         │
│        导入你的第一本书                  │
│    拖拽 EPUB/PDF 文件到此处              │
│        或点击选择文件                    │
│                                         │
│        [ 选择文件 ]                      │
│                                         │
└─────────────────────────────────────────┘
```

- 空状态居中展示，引导用户导入第一本书
- 支持拖拽文件到窗口任意位置
- "选择文件"按钮触发系统文件选择器（过滤 .epub / .pdf）

### 3.2 有书状态（封面网格）

```
┌─────────────────────────────────────────────────────┐
│  [搜索或筛选                    ]  [导入 +]         │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐          │
│  │          │  │          │  │          │          │
│  │  封面图   │  │  封面图   │  │  封面图   │          │
│  │          │  │          │  │          │          │
│  │          │  │          │  │          │          │
│  └──────────┘  └──────────┘  └──────────┘          │
│   三体         活着          百年孤独               │
│   刘慈欣       余华          马尔克斯               │
│   ████░░ 76%   ████░░ 34%   ██████ 100% ✓         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**封面卡片规格**：
- 尺寸：160×220px，圆角 8px
- Hover 状态：轻微放大（scale 1.03）+ 阴影，过渡 150ms
- 封面图片：有则显示，无则用默认占位图（书名首字 + 纯色背景）
- 底部信息：书名（单行，超出截断 + 省略号）、作者（次级色 `rgb(0x888888)`）、进度条（4px 高）+ 百分比
- 已完成的书：进度条变绿色（`rgb(0x4caf50)`），末尾打 ✓
- 右键菜单：继续阅读、删除、导出笔记

**顶部工具栏**：
- 左侧：搜索框（180px），占位文字"搜索书籍..."，按书名/作者实时过滤
- 右侧：导入按钮，触发系统文件选择器

**网格布局**：
- 列数自适应：`floor((内容区宽度 - 间距) / (160 + 24))`，间距 24px
- 左对齐，从上到下、从左到右排列

## 4. 阅读视图

### 4.1 整体布局

```
┌────────────────────────────────────────────────────────┐
│ ← 书库   三体  第一部·第一章              1 / 256 ▸    │
├────────────────────────────────────────────────────────┤
│                                                        │
│         文化大革命如火如荼地进行。                     │
│         天体物理学家叶文洁在红岸基地...                 │
│                                                        │
│         [选中的文本会高亮，弹出操作菜单]                 │
│                                                        │
├────────────────────────────────────────────────────────┤
│  ████████░░░░░░░░░░░░░░░ 35%        第1章 / 共34章     │
└────────────────────────────────────────────────────────┘
```

### 4.2 顶部工具栏

高度 44px，`bg(secondary)`，阅读时始终可见。

| 位置 | 内容 | 行为 |
|------|------|------|
| 左 | `← 书库` 按钮 | 返回书库，自动保存当前阅读位置 |
| 中左 | 书名 | 只读文本 |
| 中 | `第一部·第一章` | 章节名，点击弹出章节跳转下拉菜单 |
| 右 | `1 / 256` | 当前页/总页数，点击弹出页号输入框 |

### 4.3 阅读区域

- **EPUB**：流式文本渲染。字体大小可在设置中调整（默认 16px），行高 1.8，左右页边距 80px
- **PDF**：Canvas 图片渲染模式。支持缩放（Ctrl+滚轮）和分页
- **键盘翻页**：← → 或 PageUp / PageDown
- **阅读位置记忆**：翻页时自动保存（章节 + 页码 + 页面纯文本快照）。关闭应用或返回书库后再打开，自动恢复到上次位置

### 4.4 文字选中交互

选中文字后弹出浮动操作菜单，三项垂直排列：

```
┌─────────────┐
│ 📖 查询词义  │  → 调用 AI 解释词汇，结果在右侧对话面板显示
│ 💬 发起讨论  │  → 展开右侧对话面板，以选中文字为上下文
│ 🖊 划线标注   │  → 高亮选中文字（默认黄色），持久化到 Notes 表
└─────────────┘
```

### 4.5 底部状态栏

高度 28px，阅读时始终可见。

- 左侧：进度条（4px 高，全宽），显示 `当前页/总页数 百分比`
- 右侧：`第X章 / 共Y章`
- 进度条可点击：点击跳转到对应位置

## 5. AI 推荐讨论点

AI 在后台分析当前页文本后，在阅读区**直接标注**值得讨论的句子，作为"一键开聊"入口。

```
┌───────────────────────────────────────────┐
│ 给岁月以文明，而不是给文明以岁月。            │  ← AI推荐讨论点
│                                   💬 聊聊   │    (紫色虚线底)
└───────────────────────────────────────────┘
```

### 5.1 三种高亮状态

| 状态 | 含义 | 视觉 | 交互 |
|------|------|------|------|
| AI 推荐讨论点 | AI 圈出的值得讨论的句子 | 淡紫色（`rgba(156, 39, 176, 0.15)`）虚线边框底 + 右端 `💬 聊聊` 标签 | 点击 `💬 聊聊` → 右侧对话面板展开，以该句为上下文发起新对话 |
| 已聊过 | 用户点击过"聊聊"并产生对话 | 淡紫色实色底，标签消失 | 点击高亮文本 → 对话面板切到该句相关的历史对话 |
| 用户手动划线 | 用户选中文字点"划线标注" | 黄色（`rgba(255, 235, 59, 0.3)`）/ 绿色 / 蓝色实色底 | 点击弹出：[查看对话] [删除] [改颜色] |

### 5.2 生成时机

- 用户翻到新页面 → AI 后台异步分析 → 返回 0~3 个推荐讨论点
- **仅在对话面板已展开时生成**（用户有对话意图时才推荐，不打扰纯阅读）
- 不需要讨论的页面不做任何标记

### 5.3 讨论入口的三条路径

```
触发方式                       → 结果
──────────────────────────────────────────────
用户手动选中文字 → "发起讨论"     → 以选中文字为上下文，打开对话
用户点击 AI 推荐点 → "💬 聊聊"   → 以该句为上下文，打开对话
用户在对话面板中直接输入          → 以当前页为上下文，继续对话
```

## 6. 对话面板

右侧栏，320px 宽，全高度。宽度动画 200ms。

### 6.1 面板结构

```
┌──────────────────────────────┐
│  💬 第三章讨论        ✕  —  │  ← 标题栏
├──────────────────────────────┤
│                              │
│  [对话历史区域，可滚动]       │
│                              │
│  ┌──────────────────────┐   │
│  │ 👤 你                │   │
│  │ 叶文洁为什么要按下     │   │
│  │ 那个按钮？            │   │
│  └──────────────────────┘   │
│                              │
│  ┌──────────────────────┐   │
│  │ 📖 AI (教授模式)      │   │
│  │ 从她之前的经历来看…    │   │
│  │                      │   │
│  │ ⚑ 举报剧透           │   │
│  └──────────────────────┘   │
│                              │
├──────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 输入你的想法...          │ │
│ └─────────────────────────┘ │
│ [发送]     当前: 教授模式    │
└──────────────────────────────┘
```

### 6.2 面板行为

| 场景 | 行为 |
|------|------|
| 无对话时 | 面板隐藏（宽度 0），主内容区占满 |
| 选中文字点"发起讨论" / 点击"💬 聊聊" | 面板从右侧滑入（0→320px，200ms），主内容区同步收缩 |
| 同一本书继续对话 | 恢复上次对话历史和上下文 |
| 关闭（✕） | 收起面板，对话持久化保存 |
| 最小化（—） | 面板收起为 4px 竖条（视觉提示有活跃对话），点击竖条恢复 |

### 6.3 消息气泡

- **用户消息**：右对齐，浅色底（`bg(surface)`），圆角 12px，最大宽 85%
- **AI 消息**：左对齐，较深底（`bg(secondary)`），圆角 12px，最大宽 85%
- 每条消息头顶：角色标签（`👤 你` / `📖 AI`）+ 人格模式标签（`书友模式` / `教授模式`）
- 每条 AI 消息右下角：`⚑ 举报剧透` 链接（小字 `rgb(0x888888)`），点击后将消息标记 `spoiler_report`
- AI 正在生成时：底部显示闪烁光标动画

### 6.4 输入区（底部固定）

- 多行文本输入框，`placeholder: "输入你的想法..."`
- Enter 发送，Shift+Enter 换行
- 发送按钮在输入框右侧
- 左下角：当前人格模式标签，可点击手动切换
- 对话语言跟随设置

### 6.5 对话命名

首次发起讨论时自动生成标题（如"第三章讨论"），用户可在标题栏点击重命名。

## 9. 视觉风格

### 9.1 主题

跟随系统深色/浅色模式自动切换，不预设特定情绪色调。

| 模式 | 主背景 | 次级背景 | 文本色 | 强调色 |
|------|--------|----------|--------|--------|
| 深色 | `rgb(0x1e1e1e)` | `rgb(0x2d2d2d)` | `rgb(0xe0e0e0)` | `rgb(0x007acc)` |
| 浅色 | `rgb(0xffffff)` | `rgb(0xf5f5f5)` | `rgb(0x1a1a1a)` | `rgb(0x007acc)` |

### 9.2 字体

- 系统默认字体（San Francisco / Segoe UI / Noto Sans）
- 阅读区等宽字体可配置
- 字号：正文 16px（可调），UI 文本 13px，标题 15px

### 9.3 圆角与间距

- 卡片/面板圆角：8px
- 按钮/输入框圆角：6px
- 内容区间距：16px
- 组件内间距：8px/12px

## 10. 设计决策记录

| 决策 | 选项 | 选择 | 理由 |
|------|------|------|------|
| 对话面板位置 | 右侧栏 / 底部抽屉 / 浮窗 | 右侧栏 | 用户偏好，不遮挡阅读文字流 |
| 视觉风格 | 深色 / 暖色 / 跟随系统 | 跟随系统 | 用户偏好 |
| 书库展示 | 网格 / 列表 / 两者结合 | 封面网格 | 用户偏好，书架实物感 |
| 视图切换 | Tab栏 / 单一视图 / 左侧导航 | 左侧导航 | 用户偏好 |
| 对话显隐 | 始终可见 / 有对话才出现 / 结合 | 有活跃对话保持可见，无则隐藏 | 兼顾沉浸与便捷 |
| AI 推荐讨论点 | 在对话面板中 / 在阅读区 | 在阅读区 | 用户偏好，直接在文本上下文 |
| 总体布局 | 三栏 / 沉浸切换 / 可收缩+浮窗 | 三栏式 | GPUI 实现成本最低，结构清晰 |
